# NameMC Extras (v1)

A powerful browser extension that enhances your NameMC experience with additional features and improved functionality.

## üöÄ Installation

Install the Chromium version from the [Chrome Web Store](https://chrome.google.com/webstore/detail/namemc-extras-beta/cpphfojhgkppblcdhjbcploikanimgek)!
